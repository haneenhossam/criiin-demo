# CRIIN React V6 — Operational Enhanced

This version applies the requested operational edits:

- Page 1: doable diameter per waste, high-pollution threshold, distance threshold, correct units, P1/P2/P3 diesel-benefit logic, Greenfield/Rooftop filtering.
- Live Pollution Map: clickable site, highlighted pollution source, source distance, real OpenStreetMap frame from coordinates.
- Who We Are & Knowledge moved to the beginning.
- Waste Intelligence: site dropdown, waste %, distance to pollution, product cost, on-wheel cost, ROI, hardware link, digital twin shortcut.
- AI Recommendations: select top N sites and calculate total cost, ROI, diesel liters, energy, and on-wheel option.
- Digital Twin: Run Simulation updates KPIs and accepts site shortcut.
- Extra Time Series page for per-site and overall trend.
- Site Monitoring Actions page for updates and action tracking.
- Hardware Catalog: product placeholders, sensor details, cost method, on-wheel concept.

## Run
```bash
cd criin_react_v6_operational_enhanced
npm install
npm run dev
```

PowerShell fallback:
```bash
npm.cmd install
npm.cmd run dev
```
